# apps/web

WebApp/PWA premium estilo streaming.

## Recursos do MVP

- Login por email
- Home com secoes tipo streaming
- Cards de produto liberado/bloqueado
- Botao desbloquear integrado ao checkout da GG
- Instalacao PWA (Add to Home Screen)

## Configuracao

Defina URL da API em `public/config.json`:

```json
{
  "apiBase": "http://localhost:4000"
}
```

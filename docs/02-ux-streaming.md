# 02 - UX Streaming Premium

## Principios

- Rapido para entender em menos de 5 segundos
- Visual premium com foco em consumo e progresso
- Interacao orientada a "continuar de onde parou"

## Mapa de telas (Web/PWA)

1. Login / Criar conta
2. Home
3. Catalogo completo
4. Detalhe do produto
5. Player / Conteudo
6. Minha conta
7. Checkout / Upgrade

## Home (estilo streaming)

Secoes recomendadas:

- Continue de onde parou
- Liberados para voce
- Recomendado para seu momento
- Bloqueados com maior aderencia (com CTA)

## Cartoes de produto

- Thumbnail forte
- Badge: Liberado / Bloqueado / Novo
- Progresso (ex: 32%)
- CTA: Entrar ou Desbloquear

## Experiencias-chave para LTV

- Recomendacoes baseadas no que ja comprou
- Upsell contextual dentro da trilha
- Oferta com 1 clique (sem perder sessao)

# 11 - Design System (direcao inicial)

## Direcao visual

- Premium, clean, cinematica
- Contraste alto e tipografia marcante
- Cards com destaque para conteudo

## Tokens (base)

- Cores primarias: neutros escuros + acento quente
- Radius: medio para cards e botao
- Elevation: sombras suaves por profundidade
- Motion: transicoes curtas (150-250ms)

## Componentes MVP

1. Navbar
2. Hero rotativo
3. ProductCard
4. RowCarousel
5. ProgressPill
6. LockedOverlay
7. CTAButton
8. VideoPlayerShell

## Regras de usabilidade

- CTA principal sempre visivel
- Home com leitura rapida em mobile
- Estados vazios claros (sem confundir usuario)
